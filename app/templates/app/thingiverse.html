How to use Thingiverse

  • Enter some search criteria for the item you want(upper right)
  • Click on an item to get more info on it
  • if you want to print the item, use the "thing files" link and click on the .stl file you want. This will download the file.
  • When requesting a print, select the file you just downloaded.
  • Note that item size can also be scaled up or down as you desire, just use the scale checkbox when selecting options.
